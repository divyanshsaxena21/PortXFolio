# 🌐 PortXFolio: 3D Developer Portfolio with Three.js + React + Vite

Welcome to our submission for the **PortXFolio Hackathon** — a creative and technically advanced developer portfolio built using **React**, **Vite**, and **Three.js**. This project reimagines the traditional portfolio by integrating **interactive 3D elements**, **dynamic components**, and a smooth, performance-optimized UI.

## 🚀 Project Highlights

- **🌟 3D Experience**: Crafted with [Three.js](https://threejs.org/), this portfolio features immersive 3D scenes that bring my work and skills to life.
- **⚛️ React + Vite Stack**: Built with the lightning-fast Vite bundler for optimal development experience and performance.
- **🎯 Responsive & Interactive**: Fully responsive across devices with smooth animations and engaging interactions.
- **🧠 Tech Showcase**: Includes modern web practices—modular components, reusable hooks, optimized asset loading, and code-splitting.

## 📂 Getting Started

To run this portfolio locally:

```bash
# Clone the repo
git clone https://github.com/your-username/portxfolio-3d-portfolio.git
cd portxfolio-3d-portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

> Make sure you have Node.js ≥ 16 and a modern browser (preferably Chrome or Firefox).
